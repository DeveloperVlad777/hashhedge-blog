HASH HEDGE — БЛОГ / СТАТЬИ
================================

Структура папки
---------------
hashhedge-blog/
├── styles.css                     ← общий файл стилей (один на все статьи)
├── README.txt                     ← этот файл (список статей)
└── stop-loss-prop-traders/        ← папка статьи
    ├── ru.html                    ← русская версия
    ├── en.html                    ← английская версия
    └── pt.html                    ← португальская версия

Как это работает
----------------
• styles.css лежит в корне и общий для всех статей.
• Каждая статья — отдельная подпапка с тремя языковыми файлами (ru / en / pt).
• HTML-файлы подключают стили по пути ../styles.css (на уровень выше).
• Чтобы добавить новую статью — создаётся новая подпапка рядом со
  stop-loss-prop-traders, файлы внутри так же ссылаются на ../styles.css.
