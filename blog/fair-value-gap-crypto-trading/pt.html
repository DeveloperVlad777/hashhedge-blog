O que é um Fair Value Gap?
O termo Fair Value Gap (FVG) aparece em praticamente todo vídeo sobre Smart Money Concept (SMC) ou ICT trading. Mas a maioria das explicações usa exemplos de Forex, principalmente EUR/USD. Em cripto, a mecânica é a mesma, mas os detalhes importam. BTC e ETH se comportam de forma diferente dos pares tradicionais de moedas, e entender essas diferenças pode melhorar muito suas decisões de trading.
Neste guia, vamos explicar o que realmente é um Fair Value Gap, como a formação de três candles aparece no gráfico BTC/USDT, por que o preço tende a voltar para preencher o gap e como montar uma estratégia de trading com FVG respeitando regras de gerenciamento de risco.
Um Fair Value Gap não é um padrão gráfico tradicional. Ele representa um desequilíbrio de preço — uma área onde o preço se moveu tão rápido que poucas negociações aconteceram entre compradores e vendedores. Como o mercado tende a buscar eficiência, o preço frequentemente revisita essas zonas para "preencher o gap", permitindo que ordens que ficaram para trás sejam executadas.
Por isso, um FVG é muito mais do que um padrão visual que "funciona porque todo mundo vê". Ele reflete a mecânica real de execução de ordens. Um movimento desequilibrado cria uma área de preço ineficiente, que muitas vezes atrai o preço de volta com uma probabilidade relativamente alta.
Na prática, FVGs geralmente aparecem depois de movimentos impulsivos fortes, como: liquidity sweeps, grandes notícias, rompimentos rápidos e períodos de baixa liquidez, como finais de semana ou sessão asiática.
Como identificar um Fair Value Gap
Um FVG é identificado usando três candles consecutivos, sendo o candle do meio o movimento impulsivo.
FVG bullish: Mínima do Candle 3 > Máxima do Candle 1. O espaço entre a máxima do Candle 1 e a mínima do Candle 3 se torna o Fair Value Gap.
FVG bearish: Máxima do Candle 3 < Mínima do Candle 1. O espaço entre a mínima do Candle 1 e a máxima do Candle 3 se torna o Fair Value Gap bearish.

A zona entre 67.200 e 68.400 é o Fair Value Gap bullish. Durante a próxima correção, o preço costuma revisitar essa área antes de continuar subindo.
Um Fair Value Gap bullish se forma durante um impulso de alta. O candle do meio é um candle grande de expansão bullish, deixando um desequilíbrio abaixo dele. Quando o preço volta para dentro desse gap, traders observam possíveis oportunidades de compra.
Um Fair Value Gap bearish funciona de forma oposta. Ele se forma durante um forte movimento de baixa. O candle impulsivo é bearish e deixa um desequilíbrio acima dele. Quando o preço retorna para o gap, a zona pode se tornar uma possível área de entrada short.
Um ponto importante para traders de cripto: nem todos os FVGs têm o mesmo peso. Um FVG formado depois de um liquidity sweep no timeframe H4 costuma ser muito mais relevante do que um FVG criado aleatoriamente em um gráfico M5.
Quais timeframes funcionam melhor para FVG no BTC e ETH?
Para BTC/USDT, os Fair Value Gaps de maior qualidade geralmente aparecem nos gráficos H4 e Diário. Esses timeframes maiores costumam representar fluxo institucional real, aumentando a probabilidade de o preço revisitar a zona.
Para ETH/USDT, tanto H1 quanto H4 produzem zonas de FVG confiáveis. Embora FVGs também apareçam no M15 e em timeframes menores, o ruído do mercado aumenta muito, reduzindo a qualidade do sinal.
Um fluxo prático: identifique o Fair Value Gap no H4 ou Diário, espere o preço retornar e refine a entrada no H1 ou M15. Essa abordagem multi-timeframe oferece um equilíbrio melhor entre precisão e confiabilidade.
Por que o preço volta para preencher um Fair Value Gap?
O mercado é, na essência, um sistema de ordens. Um movimento impulsivo acontece quando um lado do mercado domina o outro com compras ou vendas agressivas. Como resultado, o preço se move tão rápido que muitas ordens ficam sem execução dentro de uma determinada faixa de preço. Market makers e sistemas algorítmicos frequentemente levam o preço de volta para essas áreas para executar ordens restantes. Isso não é "mágica" do mercado — é simplesmente como a execução de ordens funciona.
Quando os Fair Value Gaps são preenchidos? A maioria dos Fair Value Gaps acaba sendo preenchida, mas nem todos. Durante uma tendência diária forte, um FVG no H4 pode ficar intocado por dias ou até semanas. Gaps formados depois de eventos estruturais importantes — como o rompimento de uma resistência de vários meses ou uma reação a uma notícia macro relevante — geralmente são preenchidos com menos frequência. Regra prática: um Fair Value Gap no H4 formado durante um range ou movimento corretivo tem uma probabilidade muito maior de ser preenchido do que um FVG no H4 criado no início de uma nova tendência forte.
Preenchimento parcial vs. preenchimento completo. O preço não precisa atravessar todo o Fair Value Gap. Em muitos casos, apenas tocar a borda superior ou inferior da zona já é suficiente antes de a tendência continuar. Isso é conhecido como preenchimento parcial. Um preenchimento completo acontece quando o preço atravessa todo o desequilíbrio e fecha além do lado oposto do gap. Para trading, o início do preenchimento costuma ser a parte mais importante — é ali que os traders buscam entradas. O objetivo não é necessariamente esperar o preço preencher todo o gap, mas identificar um ponto de continuação de alta probabilidade perto do início do desequilíbrio.

Como operar um Fair Value Gap
Espere o preço retornar antes de entrar. Um dos erros mais comuns é comprar imediatamente após a formação de um FVG, simplesmente seguindo o movimento impulsivo. Uma abordagem melhor é esperar o preço corrigir para dentro do Fair Value Gap antes de considerar uma entrada. Exemplo: se o BTC/USDT forma um FVG bullish depois de romper uma resistência, espere o preço voltar para dentro do desequilíbrio. Quando o preço entra no gap, procure confirmação de que compradores estão voltando antes de abrir uma posição long. Essa abordagem melhora bastante a relação risco-retorno, porque a entrada acontece mais perto do nível de invalidação e ainda mantém um alvo maior para cima.
Onde colocar o stop-loss? Em um setup bullish, o stop-loss deve ficar abaixo do Fair Value Gap. Em um setup bearish, o stop deve ficar acima do Fair Value Gap. A lógica é simples: se o preço atravessa completamente o desequilíbrio e fecha além dele, o Fair Value Gap falhou como zona de suporte ou resistência. Nesse ponto, o setup foi invalidado, e o trade deve ser encerrado conforme seu plano.

Combinando FVG com estrutura de mercado
Fair Value Gaps se tornam muito mais confiáveis quando combinados com estrutura de mercado. Um FVG bullish formado acima do Higher High anterior tem muito mais peso do que um FVG localizado no meio de um range. Da mesma forma, um FVG bearish formado abaixo do Lower Low anterior oferece confirmação mais forte de continuação da tendência de baixa.
Isso é especialmente verdadeiro no ETH/USDT durante mercados fortemente direcionais. Uma sequência de FVGs bullish se formando progressivamente mais acima — ou FVGs bearish se formando cada vez mais abaixo — geralmente indica pressão institucional sustentada de compra ou venda. Entender a relação entre Fair Value Gaps e estrutura de mercado ajuda traders a diferenciar setups de alta qualidade de ruído comum do mercado.
Fair Value Gaps e false breakouts
Fair Value Gaps estão diretamente conectados a false breakouts. Quando o preço faz um liquidity grab ou liquidity sweep e depois reverte, essa reversão quase sempre deixa um Fair Value Gap para trás. Isso explica por que false breakouts e liquidity grabs frequentemente criam FVGs — eles são uma das bases da análise Smart Money.
Por que liquidity grabs costumam deixar um FVG. Um liquidity grab é um movimento rápido e agressivo. Quando o preço dá um spike além de um nível-chave por um ou dois candles antes de reverter, o candle impulsivo da reversão quase sempre cria um Fair Value Gap. Esse desequilíbrio é uma evidência de acumulação institucional de ordens. Grandes participantes usam traders de rompimento do varejo como liquidez para montar suas próprias posições. No BTC/USDT, depois de um grande liquidity sweep no gráfico H4, o impulso de reversão frequentemente deixa um Fair Value Gap no H1. O preço muitas vezes revisita esse desequilíbrio antes de continuar na nova direção.
Usando FVG como confirmação após um stop hunt. Uma sequência clássica de Smart Money: um liquidity sweep remove stop-losses agrupados; a reversão cria um Fair Value Gap; o preço se afasta do sweep; o preço retorna ao Fair Value Gap para um reteste; a tendência continua. Entrar nesse reteste é considerado um dos setups de maior probabilidade dentro da metodologia Smart Money. Em um setup bullish, o Fair Value Gap deve se formar acima do liquidity sweep; em um setup bearish, deve se formar abaixo. Isso confirma que compra ou venda institucional já entrou no mercado.
Operando Fair Value Gaps em uma conta de prop trading
Por que entradas em FVG combinam melhor com prop firms do que breakout trading. Operar diretamente a partir de um Fair Value Gap normalmente envolve uma ordem limit ou uma entrada após um pullback controlado. Diferente de perseguir candles de rompimento, um setup de FVG oferece: entrada claramente definida, stop-loss lógico e alvo mensurável. Essas características tornam estratégias com Fair Value Gap muito mais compatíveis com as regras rígidas de gerenciamento de risco usadas por prop firms.

O que fazer quando um FVG aparece perto do seu limite de drawdown diário. Imagine que você está chegando ao fim do dia e seu drawdown já está em 3–4%, enquanto seu limite diário de perda é 5%. Um Fair Value Gap aparentemente perfeito aparece no gráfico. A decisão correta é não entrar no trade. O tamanho da posição deve sempre ser definido antes da entrada, com base no risco pré-determinado por trade. Se a margem restante até o limite diário de drawdown for menor do que o risco planejado para a operação, simplesmente pule o setup. O Fair Value Gap não vai desaparecer. Se o desequilíbrio continuar sem ser preenchido na próxima sessão, o setup ainda pode ser válido amanhã. Traders profissionais protegem o capital primeiro e procuram oportunidades depois.
Principais conclusões
✓
Um Fair Value Gap (FVG) é um desequilíbrio de preço formado entre a máxima do primeiro candle e a mínima do terceiro candle, no caso de um FVG bullish, ou entre a mínima do primeiro candle e a máxima do terceiro candle, no caso de um FVG bearish. O mercado tende naturalmente a revisitar essas áreas ineficientes de preço.
✓
Os FVGs de maior qualidade no mercado cripto geralmente se formam depois de um liquidity sweep ou liquidity grab. Esses desequilíbrios têm importância estrutural porque mostram onde participantes institucionais acumularam posições.
✓
Para operar FVGs de forma eficiente, combine-os com estrutura de mercado: determine a tendência geral em um timeframe maior, identifique Fair Value Gaps no H4 ou H1 e procure entradas em timeframes menores quando o preço retornar ao desequilíbrio. Essa abordagem multi-timeframe melhora muito a qualidade dos trades.
✓
Em uma conta de prop trading, entradas em Fair Value Gap oferecem pontos de entrada claros, posicionamento lógico de stop-loss e gerenciamento consistente de posição. O mais importante: nunca aumente o tamanho da posição apenas porque um setup parece excepcionalmente bom. As regras de gerenciamento de risco devem permanecer as mesmas para todos os trades. Consistência — não confiança excessiva — é o que permite que traders sobrevivam a avaliações de prop firms e construam lucratividade no longo prazo.
FAQ
O que é Fair Value Gap em termos simples?
−
Um Fair Value Gap é uma área no gráfico onde o preço se moveu rápido demais, deixando poucas ordens executadas entre compradores e vendedores. O mercado tende a retornar a essa zona para "fechar" o desequilíbrio. É identificado usando três candles: se há um gap entre a máxima do primeiro candle e a mínima do terceiro — é um FVG bullish; se entre a mínima do primeiro e a máxima do terceiro — é um FVG bearish.
Todos os Fair Value Gaps são preenchidos?
+
A maioria sim, mas nem todos. Uma tendência forte no D1 pode deixar FVGs no H4 intocados por semanas. Gaps formados após grandes rompimentos estruturais ou eventos macro significativos são preenchidos com menos frequência. Regra prática: um FVG formado dentro de um range ou correção tem muito mais probabilidade de ser preenchido do que um criado no início de uma nova tendência forte.
Quais timeframes são melhores para operar FVGs?
+
A abordagem ideal: identificar FVGs no H4 ou Diário, onde há fluxo institucional genuíno por trás do impulso, e então buscar a entrada no H1 ou M15. Isso reduz sinais falsos e fornece um local lógico para o stop-loss. FVGs no M15 e abaixo existem, mas a relação sinal-ruído é significativamente pior.
Como um FVG se relaciona com false breakouts?
+
Diretamente. Quando o preço faz um liquidity grab ou sweep e reverte, o candle de reversão quase sempre deixa um Fair Value Gap para trás. Esse é um rastro de acumulação institucional de posição. Por isso, a sequência "false breakout → FVG → reteste" é considerada um dos setups de entrada de maior probabilidade na metodologia Smart Money.
É possível usar estratégias de FVG em um prop firm challenge?
+
Sim, e os setups de FVG são particularmente adequados para prop trading: há um ponto de entrada claro, stop lógico e alvo mensurável. Diferente de perseguir rompimentos, entradas em FVG não exigem "adivinhar" o impulso. A regra principal: nunca entre se a margem restante antes do seu limite diário de drawdown for menor do que o risco planejado para o trade.
